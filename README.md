# Stash web-application
Stash is mobile focused web-app

**Smartly Organizing Your Universe.**

A personalized information management and reference application designed to help you capture, categorize, and master your notes, and digital life.

## Features

* Quickly save links, notes, and snippets from any source.
* Categorize your information with 6 different categories.
* Add notes, delete notes.
* Search fot saved notes.
* clean and UI focused 

## Getting Started

Follow these steps to get the development environment running on your local machine.

You need Node.js and npm (Node Package Manager) installed.

* [Node.js Download](https://nodejs.org/)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/leeparksara/smart-bookmark.git]
    cd [smart-bookmark]
    ```

2.  **Install dependencies:**
   open the terminal on vscode
    ```bash
    npm install
    ```

## ⚙️ Development Scripts

Use these commands to manage your local development environment.


 **Start Dev Server**  `npm run dev` , Runs the development server and opens the `loading.html` entry page. 
 **Build for Production**  `npm run build`  Compiles the project assets into the `dist/` directory for deployment. 
 **Run Tests**  `npm run test` ,  Executes any predefined tests.
